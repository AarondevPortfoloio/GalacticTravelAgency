# Galactic Travel Agency

A simple C# console application that demonstrates basic variables, type conversions, and console output.

## Description

This program simulates a galactic travel agency scenario:

- Stores passenger information: name, age, ticket type, and preferred planet.
- Demonstrates incrementing numbers, type casting, and string conversion.
- Prints all information to the console.

It is intended as a learning example for beginners in C#.

## How to Run

1. Make sure **.NET SDK** is installed. Check with:
   ```bash
   dotnet --version
   ```

2.	Open the project folder in VS Code:
    code .

3.	Build and run the program using:
    dotnet run

4.	Expected output:
    Zara
    Mars
    29
    29
    29
    29
